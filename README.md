# π Shell (psh)
A simple Unix-like shell written in C.

## Build & Run
### Requirements

A Unix-like system (Linux, macOS, WSL).
GCC (or any C compiler).

### Compile
```bash
gcc -o psh shell.c
```

### Run
```
./psh
```

### Usage Examples
```bash
π> ls -la
π> cd ..
π> pwd
π> git clone https://github.com/Reza-namvaran/p-shell
```
